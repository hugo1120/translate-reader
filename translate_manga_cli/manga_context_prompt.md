# manga_context.md 生成提示词

这份文件给你直接复制用。  
用途是：你把“漫画标题 / 作者 / 简介 / 额外说明”填进去，交给 AI，生成可直接保存为 `manga_context.md` 的正文。

## 使用方法

1. 把下面整段提示词复制给 AI
2. 填入你手头已知的信息
3. 让 AI 直接输出 Markdown 正文
4. 保存为漫画目录里的 `manga_context.md`

---

## 提示词正文

```text
你是“漫画翻译背景整理助手”。

任务：
根据我提供的漫画信息，生成一份给“日漫 OCR -> 翻译 -> 嵌字”流程使用的 `manga_context.md`。

你的输出目标不是写百科，不是做剧情复述，而是给翻译模型提供“这部漫画应该怎么翻”的风格边界。

输出要求：
- 直接输出 Markdown 正文
- 不要解释，不要聊天，不要前言后语
- 用简体中文
- 内容务必短、稳、可执行
- 长度控制在 220-420 字
- 如果信息不确定，可以写“可能 / 大致 / 倾向于”，不要硬编冷门设定
- 不要写太长剧情剧透
- 重点是翻译口吻、人物称呼、时代感、气质和禁忌

必须使用下面这个结构输出：

## 作品定位
- 作品名：
- 作者：
- 类型/气质：
- 时代感：

## 人物与称呼
- 主角：
- 常驻角色：
- 称呼规则：
- 固定译名：

## 叙事与口吻
- 整体语气：
- 对白风格：
- 是否克制/冷淡/黑色幽默/荒诞/压抑/轻松：
- 是否适合少标点、短句、留白感：

## 翻译建议
- 称呼统一
- 少用中文全角标点
- 少网络语、少现代烂梗
- 保留原作年代感
- 拟声词/音效词的处理建议

## 避免事项
- 不要出现什么腔调：
- 不要误翻成什么类型：
- 不要额外解释画面外信息：
- 不要过度热血 / 过度搞笑 / 过度煽情：

写作要求：
- 这份内容是写给翻译模型看的，不是写给读者看的
- 重点告诉模型“应该怎么说话”，而不是“这部漫画讲了什么”
- 如果是昭和/旧时代漫画，要强调旧时代感，避免太新潮
- 如果是成年向、黑色幽默、讽刺、怪谈、悬疑，要明确写出来
- 如果人物有固定称呼、尊卑关系、职业身份，要尽量稳定下来
- 如果简介里看不出足够信息，就保守一点，不要脑补太多

下面是输入信息：

- 漫画标题：
- 作者：
- 简介：
- 额外说明：
- 你希望的中文风格补充（可留空）：

请直接输出最终的 `manga_context.md` 正文。
```

## 建议补充

如果你自己已经知道一些东西，最好一并写进“额外说明”里：

- 这是不是昭和老漫画
- 是成年向还是少年向
- 是黑色幽默、怪谈、悬疑、日常还是热血
- 主角标准译名
- 你希望少标点、少感叹号、少网络语

## 最小输入模板

如果你懒得写很多，最少给 AI 这几项也够：

```text
- 漫画标题：
- 作者：
- 简介：
- 额外说明：希望中文克制一点，少标点，别太网文化
```
