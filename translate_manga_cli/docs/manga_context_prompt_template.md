# 生成 `manga_context.md` 的提示词模板

这份文档给后续 AI 用。目标是让 AI 根据作者名、书名、目录信息，生成一份适合漫画翻译器使用的 `manga_context.md`。

## 推荐提示词

```text
你是漫画背景整理助手。

任务:
根据我提供的漫画作者、书名、卷名、目录路径信息，生成一份给“日漫机翻 + 嵌字”模型使用的背景提示词文件。

输出要求:
- 直接输出 Markdown 正文
- 不要解释，不要聊天
- 用简体中文
- 长度控制在 180-320 字
- 如果信息不确定，请明确写“可能/大致”，不要编造冷门细节

必须包含这些部分:
1. 作品定位
2. 人物与称呼
3. 叙事与口吻
4. 翻译建议
5. 避免事项

写作重点:
- 不是写百科全书
- 重点是告诉翻译模型“这部漫画应该怎么说话”
- 要强调时代感、人物口吻、是否成年向、是否黑色幽默、是否压抑/荒诞/冷淡
- 尽量给出固定译名
- 标点习惯也可以写进去，比如“少感叹号, 少口语网感”

输入信息:
- 作者:
- 作品名:
- 卷名:
- 目录路径:
- 额外备注:
```

## 生成结果验收标准

- 看完后能立刻知道这部作品的语气边界
- 能约束“称呼统一”“不要乱热血”“不要过量标点”
- 不超过一屏，适合每次翻译都带进去

## 示例倾向

- 成年短篇黑色幽默:
  强调克制、冷笑、讽刺、人性缺陷
- 校园日常:
  强调轻松、口语化、自然称呼
- 昭和老漫画:
  强调旧时代感、不要太新潮网络语
